# adelbaseli.github.io

This is my personal portfolio. Here I share my experience, skills, and projects.

Live at: [adelbaseli.github.io](https://adelbaseli.github.io)

Built with Astro, React, and Tailwind CSS, based on a template by [rkeshs](https://github.com/rkeshs/my-portfolio) (MIT License, see [LICENSE](LICENSE)).

## Running it locally

```bash
git clone https://github.com/Adelbaseli/adelbaseli.github.io.git
cd adelbaseli.github.io
bun install
bun dev
```

Then open `http://localhost:4321`.
